#左右菜单栏滑动组件
***
| 属性名        | 类型          | 默认值  |说明                                     |
| ------------- |:-------------:|:-------:|----------------------------------------:|
| wrapheight    | Number        |   0px   |控制组件整体高度                         |
| activemenu    | Number        |    -1   |选中右侧菜单索引需要与activemenu2同时设置|
| activemenu2   | Number        |    -1   |选中右侧菜单索引需要与activemenu同时设置 |
| list          | Array         |    []   |显示组件内容                             |
| handlecontent | event         |         |用来处理点击右侧菜单的行为               |
